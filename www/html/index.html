About the Project

Webserv 1337 is a lightweight HTTP/1.1 server implementation built from scratch using C++98. This project demonstrates our understanding of network programming, HTTP protocols, and server architecture principles.

HTTP/1.1 Compliant

Fully implements the HTTP/1.1 protocol specifications

Multiple HTTP Methods

Supports GET, POST, and DELETE methods

CGI Support

Execute server-side scripts through Common Gateway Interface

Configuration File

Customize server behavior through simple configuration files

Error Handling

Proper error responses with custom error pages

Session Management

Cookie-based session handling for stateful applications
