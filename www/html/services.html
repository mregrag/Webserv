GET with Query Parameters

Retrieve data from the server by specifying query parameters.

Example with cURL:

curl "http://localhost:8080/data?id=123"

GET a Static Resource

Request static files directly from the server.

This form loads an HTML file directly without using query parameters.

Example with cURL:

curl http://localhost:8080/static/test_page.html

POST: Upload Files

Submit files to be stored on the server.

Example with cURL:

curl -X POST -F "file=@your_file.txt" http://localhost:8080/upload

DELETE Resources

Remove specific files or resources from the server.

Example with cURL:

curl -X DELETE http://localhost:8080/uploads/filename.txt
